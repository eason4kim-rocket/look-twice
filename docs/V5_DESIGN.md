# Look Twice v5 — Embodied Evidence Assurance

## Frozen positioning

Look Twice v5 is a **contract-gated mobile navigation and proxy-manipulation**
system under lineage-aware evidence on AMD GPU. It is not a quadruped showcase,
not a full Purify product, and not a sim-to-real claim.

```text
Claims → lineage + conformal → per-action GateReceipt
→ physical navigation / approach
→ pick_proxy under second contract
→ mid-execution PlanInvalidation
→ continue, detour, abort, or safe_fallback
```

v4 remains frozen (`v4.0-hackathon-final`). v5 lives on `v5-embodied-evidence`.

## Three capabilities

1. **Physical navigation** — skid-steer default for demos; kinematic only for batch.
2. **Simple proxy manipulation** — approach + close + lift test (no dexterous hand).
3. **Dual contracts + invalidation** — `cross_region` and `pick_proxy`; TTL/dynamic events invalidate admitted plans.

## Explicit non-goals

Quadruped/humanoid full-body control, real robot transfer claims, VLA/RL training,
multi-robot full physics, private Purify integration, rewriting v4 formal 960.
